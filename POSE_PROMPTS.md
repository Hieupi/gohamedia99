# Cách dùng

1. **Chọn loại ref phù hợp:**
   - Scene cần **pose cụ thể** (ngồi/đứng/tựa) → dùng PHẦN A
   - Scene cần **đảm bảo size vòng 1** → dùng PHẦN B
   - Muốn cả 2 → dùng PHẦN C
2. Copy prompt → paste vào Nano Banana Pro / Midjourney / Ideogram
3. Tải ảnh kết quả
4. Vào https://gohaaff.click → Phối Đồ → upload vào **"POSE tham chiếu"**
5. Generate

## Ghép nhiều lớp (max effect)

- Upload CHEST close-up vào "POSE tham chiếu"
- Kết hợp với model thật ở nút Mặt (face ref)
- Kết hợp với outfit ref
- → AI có 3 tín hiệu: identity + shape + garment → kết quả ổn định nhất

## Tips

### Chọn Pose standing:
- **POSE 1, 2, 10** — chest-forward cực mạnh (vai kéo sau tối đa, tay ra sau/lên đầu)
- **POSE 4, 8** — side-profile silhouette ngực rõ nét nhất trong profile view
- **POSE 3, 9** — commercial/catwalk standard, video-ready
- **POSE 5, 7** — editorial đa dạng cho 4 scene
- **Tất cả 10 pose đều ĐỨNG** → video-ready cho Kling AI 3.0

### Chọn Chest close-up:
- **CHEST 1** match nhất với ảnh mẫu trắng tank top
- **CHEST 3** cho commercial/studio clean look
- **CHEST 4** cho boudoir/pink silk slip

### Nếu AI tạo ref chưa đủ to:
- Thêm vào prompt: `voluptuous mature adult feminine figure, fabric visibly stretched and tensioned across full chest volume, dramatic hourglass silhouette`
- Hoặc thêm: `chest volume prominently defined in the silhouette, not reduced, not minimized`

### Nếu bị safety filter:
- "voluptuous / full bust" → "curvy silhouette / generous feminine proportions"
- "décolletage" → "neckline area"
- "cleavage" → "chest contour / V-line silhouette"
